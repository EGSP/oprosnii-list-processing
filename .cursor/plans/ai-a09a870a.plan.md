- Инициализировать SvelteKit проект с TypeScript (Node.js LTS, npm)
- Выбрать опции: TypeScript, ESLint, Prettier, Playwright (опционально)
- Проверить что проект запускается

### 1.2 Настройка структуры папок

- Создать `lib/business/` для бизнес-логики
- Создать `lib/components/` для UI компонентов
- Создать `data/tu/` для хранения JSON файлов с ТУ

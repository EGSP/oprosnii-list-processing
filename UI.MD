# Архитектура UI\UX
Весь интерфейс работает на одном экране. Без переходов на другие страницы.

## Main Layout 
Экран делится на две части: левая панель, правое окно

Левая панель содержит список загруженных заявок
Правое окно отображает информацию о выбранной заявке

### Левая панель (vertical layout)

- Кнопка загрузки заявки + возможность DragNDrop.
    Список заявок сразу обновить при успешной загрузке.
    Выбрать загруженную заявку.

1. Список загруженных заявок
    - Если заявок нет - индикация что список пуст
    - Если заявки есть - отображение заявок по порядку даты загрузки (сверху самая поздняя)

### Правое окно (Vertical layout)

1. Если заявка не выбрана - индикация отсутствия выбора
2. Если заявка выбрана - отображение параметров заявки
    - Кнопка "Обработать заявку"
    - Группа параметров
